  STUDENT MANAGEMENT SYSTEM - SETUP INSTRUCTIONS
═══════════════════════════════════════════════════════════════

🎯 IMPORTANT: Your application now uses a backend server!
   This means you can access your student data from ANY computer,
   browser, or device on your network!

═══════════════════════════════════════════════════════════════
STEP 1: INSTALL NODE.JS (One-time setup)
═══════════════════════════════════════════════════════════════

Node.js is required to run the backend server.

1. Download Node.js from: https://nodejs.org/
   - Click the green "LTS" (Long Term Support) button
   - Download will start automatically

2. Run the installer
   - Accept the default settings
   - Click "Next" through all steps
   - Installation takes about 2-3 minutes

3. Verify installation
   - Open Command Prompt or PowerShell
   - Type: node --version
   - You should see something like: v18.17.0 or v20.x.x

═══════════════════════════════════════════════════════════════
STEP 2: INSTALL DEPENDENCIES (One-time setup)
═══════════════════════════════════════════════════════════════

1. Open Command Prompt or PowerShell

2. Navigate to the project folder:
   cd C:\Users\super\student-management

3. Install required packages:
   npm install

   This will install:
   - Express (web server)
   - CORS (for cross-origin requests)
   
   Installation takes about 30 seconds.

═══════════════════════════════════════════════════════════════
STEP 3: START THE SERVER (Every time you use the app)
═══════════════════════════════════════════════════════════════

1. Open Command Prompt or PowerShell

2. Navigate to the project folder (if not already there):
   cd C:\Users\super\student-management

3. Start the server:
   npm start

4. You should see:
   ╔════════════════════════════════════════════════════╗
   ║   Student Management System - Server Running      ║
   ╠════════════════════════════════════════════════════╣
   ║   URL: http://localhost:3000                      ║
   ║   API: http://localhost:3000/api/students         ║
   ║                                                    ║
   ║   Press Ctrl+C to stop the server                 ║
   ╚════════════════════════════════════════════════════╝

5. Open your browser and go to:
   http://localhost:3000

6. The application is now running!

═══════════════════════════════════════════════════════════════
ACCESSING FROM OTHER COMPUTERS/DEVICES
═══════════════════════════════════════════════════════════════

To access the application from other devices on your network:

1. Find your computer's IP address:
   - Open Command Prompt
   - Type: ipconfig
   - Look for "IPv4 Address" (usually starts with 192.168.x.x)
   - Example: 192.168.1.105

2. On the other device:
   - Open a web browser
   - Go to: http://YOUR-IP-ADDRESS:3000
   - Example: http://192.168.1.105:3000

3. Both devices will see the same student data!

Note: Your server computer must be running for other devices to access it.

═══════════════════════════════════════════════════════════════
STOPPING THE SERVER
═══════════════════════════════════════════════════════════════

To stop the server:
- In the Command Prompt/PowerShell window where the server is running
- Press: Ctrl + C
- Confirm if asked

═══════════════════════════════════════════════════════════════
TROUBLESHOOTING
═══════════════════════════════════════════════════════════════

❌ "npm is not recognized"
   → Node.js is not installed. Go back to Step 1.

❌ "Port 3000 is already in use"
   → Another application is using port 3000
   → Solution: Stop that application or change the port in server.js

❌ "Cannot connect from other device"
   → Check if both devices are on the same network
   → Check your firewall settings (may need to allow port 3000)
   → Make sure the server is still running on the host computer

❌ "Failed to load students"
   → The server is not running
   → Solution: Make sure to run "npm start" before opening the browser

DATA LOCATION
═══════════════════════════════════════════════════════════════

C:\Users\super\student-management\students-data.json

To restore data:
- Replace students-data.json with your backup file
- Restart the server
